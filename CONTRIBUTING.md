## Contributing to ngx-products-3d

We welcome contributions to the ngx-products-3d library.

### Getting Started

1. Fork the repository on GitHub.
2. Clone your fork locally.
3. Install dependencies: `npm install`
4. Run the playground: `npm start`
5. Build the library: `npm run build:lib`

### Making Changes

1. Create a feature branch.
2. Make your changes following existing code style (tabs, single quotes).
3. Verify the library builds and the playground runs.

### Submitting Changes

1. Commit with a clear message.
2. Push to your fork and open a pull request against `main`.
3. Bump `version` in `projects/ngx-products-3d/package.json` only when preparing a release.

### Releases

Releases are automated when a version bump is merged to `main`. See `.github/workflows/release-publish.yml`.

Thank you for contributing!
